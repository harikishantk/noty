# Noty - Next.js Project

A modern [Next.js](https://nextjs.org) 16.2.4 project fully initialized with TypeScript, Tailwind CSS v4, and ESLint.

## Project Setup

This project was initialized with the following configuration:
- **Next.js**: 16.2.4
- **React**: 19.2.4
- **TypeScript**: v5
- **Tailwind CSS**: v4
- **ESLint**: v9
- **Node.js**: v20.11.0
- **npm**: 10.2.4

All 358 dependencies installed with zero vulnerabilities.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The page auto-updates as you edit `app/page.tsx`.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm start` - Run the production build
- `npm run lint` - Run ESLint code quality checks
